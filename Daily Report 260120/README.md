# Daily Report 260120 (Math4U)

본 워크플로우는 Math4U 수업 관리 시스템의 핵심 자동화 프로세스로, 노션(Notion)에 기록된 수업 데이터를 바탕으로 학부모에게 수업 리포트를 발송하고, 특정 조건 충족 시 수업료 납부 알림을 자동으로 전송합니다.

---

## 🔄 워크플로우 흐름

1.  **Webhook**: 노션 데이터 변경 또는 특정 이벤트 발생 시 데이터를 수신합니다.
2.  **Notion - Student_Study 재조회**: 수신된 ID를 바탕으로 해당 수업 상세 정보를 조회합니다.
3.  **Notion - Students 조회**: 수업에 연결된 학생의 기본 정보(이름, 연락처 등)를 가져옵니다.
4.  **Notion - Class_Study 조회**: 수업 세부 정보 및 연관된 '지난 수업' 데이터를 조회합니다.
5.  **Notion - 지난수업 조회**: 이전 수업의 과제 내용을 확인하여 리포트에 포함합니다.
6.  **Code**: 수집된 데이터를 가공하여 수업 리포트 및 납부 알림 메시지 본문을 생성합니다.
    *   수업 회차와 수업 구분 번호를 비교하여 납부 알림 발송 여부를 결정합니다.
7.  **HTTP Request - 수업 리포트**: 가공된 데이터를 바탕으로 알림톡(또는 메시지)을 발송합니다.
8.  **If - 납부 알림 조건 체크**: 수업료 납부 대상 여부를 판별합니다.
    *   **True**: HTTP Request를 통해 납부 알림을 발송하고, 노션에 발송 완료 상태를 업데이트합니다.
    *   **False**: 별도의 알림 없이 노션에 발송 완료 처리만 수행합니다.

## ⚙️ 주요 기능 및 로직

*   **수업 리포트 자동화**: 수업 내용, 지난 과제, 수업 일시 등을 정리하여 학부모에게 발송합니다.
*   **납부 알림 자동화**: 수업 회차와 수업 구분(과정)의 숫자가 일치할 경우, 자동으로 수업료 납부 안내 메시지를 생성하여 발송합니다.
*   **연락처 관리**: 학부모 연락처(모/부)를 모두 확인하여 등록된 번호로 메시지를 발송합니다.
