# DCT 트렌드 기반 블로그 자동작성 워크플로우

본 워크플로우는 매주 일요일 오전 9시에 네이버 쇼핑 트렌드를 분석하고, Gemini AI를 활용하여 판촉/홍보/답례품 추천 블로그 포스팅을 자동으로 생성한 뒤 노션(Notion) 데이터베이스에 저장하는 자동화 시스템입니다.

---

## 📋 개요
- **목적**: 시즌별 트렌드에 맞는 판촉물 추천 콘텐츠 자동화
- **실행 주기**: 매주 일요일 오전 09:00
- **주요 기술**: n8n, Naver DataLab API, Google Gemini API, Notion API

---

## 🔄 워크플로우 흐름
1. **Schedule Trigger**: 매주 일요일 오전 9시 자동 실행
2. **현재 시기 정보**: 날짜를 계산하여 계절, 이벤트, API 요청용 날짜 범위 생성
3. **네이버 트렌드 수집**: 네이버 데이터랩 API를 통해 판촉물/생활용품/답례품 카테고리 검색량 수집
4. **시기+트렌드 합치기**: 수집된 데이터와 시기 정보를 결합하여 AI 프롬프트용 데이터 생성
5. **Gemini 블로그 작성**: Gemini AI가 트렌드 기반의 블로그 초안 작성
6. **블로그 추출**: JSON 파싱 및 노션 API 제한에 맞춘 텍스트 청크 분할
7. **노션 페이로드 준비**: 노션 블록 구조에 맞게 데이터 변환
8. **노션 저장**: 노션 DB에 최종 포스팅 저장

- **프롬프트**: DCT컴퍼니 마케터 페르소나를 부여하여 SEO 최적화 및 이미지 설명 포함 작성

- **GEMINI_API_KEY**: n8n 서버 환경변수에 설정되어 있어야 합니다.
