# DCT 트렌드 기반 블로그 자동작성 워크플로우

본 워크플로우는 매주 일요일 오전 9시에 네이버 쇼핑 트렌드를 분석하고, Gemini AI를 활용하여 판촉/홍보/답례품 추천 블로그 포스팅을 자동으로 생성한 뒤 노션(Notion)에 저장하는 자동화 시스템입니다.

---

## 📋 개요
- **목적**: 시즌별 트렌드에 최적화된 판촉물 추천 콘텐츠 자동 생성
- **실행 주기**: 매주 일요일 오전 9시
- **주요 기술**: n8n, Naver DataLab API, Google Gemini API, Notion API

---

## 🔄 워크플로우 흐름
1. **Schedule Trigger**: 매주 일요일 오전 9시 자동 실행
2. **현재 시기 정보**: 현재 날짜를 기준으로 계절, 이벤트, API용 날짜 범위 계산
3. **네이버 트렌드 수집**: 네이버 데이터랩 API를 통해 판촉물/생활용품/답례품 카테고리 검색량 분석
4. **데이터 병합**: 시기 정보와 트렌드 데이터를 결합하여 프롬프트 생성
5. **Gemini 블로그 작성**: AI가 트렌드 기반의 블로그 초안 작성
6. **데이터 파싱 및 분할**: JSON 파싱 및 노션 글자 수 제한(2,000자)에 맞춘 본문 청크 분할
7. **노션 저장**: 노션 데이터베이스에 최종 포스팅 저장

## 🛠️ 트러블슈팅

- **네이버 API 401 오류**: API 키 만료 또는 오입력 확인
- **Gemini API 429 오류**: API 호출 한도 초과. 잠시 후 재실행하거나 Google AI Studio에서 할당량 확인
- **노션 404 오류**: 노션 DB에 인테그레이션이 연결되지 않았을 경우 발생. DB 설정에서 연결 확인
- **노션 400 오류 (글자 수 제한)**: 본문이 2,000자를 초과할 경우 발생. '블로그 추출' 노드에서 분할 로직 확인
