# DCT 트렌드 기반 블로그 자동작성 워크플로우

본 워크플로우는 매주 일요일 오전 9시에 네이버 쇼핑 트렌드를 분석하고, Gemini AI를 활용하여 판촉물, 홍보물, 답례품 추천 블로그 포스팅을 자동으로 생성하여 노션(Notion)에 저장하는 자동화 시스템입니다.

---

## 📋 개요
- **목적**: 시기별 트렌드를 반영한 마케팅 콘텐츠 자동화
- **실행 주기**: 매주 일요일 오전 9시 (Schedule Trigger)
- **주요 기능**:
    - 현재 시기(계절/이벤트) 분석
    - 네이버 데이터랩 쇼핑 인사이트 API를 통한 검색 트렌드 수집
    - Gemini AI를 활용한 블로그 초안 작성
    - 노션 DB 자동 업로드

---

## 🔄 워크플로우 흐름
1. **Schedule Trigger**: 매주 일요일 9시 자동 실행
2. **현재 시기 정보**: 날짜 기반 계절 및 특수 이벤트 판단
3. **네이버 트렌드 수집**: 쇼핑인사이트 API를 통해 판촉물/답례품 카테고리 데이터 수집
4. **데이터 병합**: 시기 정보와 트렌드 데이터를 결합하여 프롬프트 생성
5. **Gemini AI 호출**: 수집된 데이터를 바탕으로 블로그 포스팅 초안 작성
6. **블로그 추출 및 파싱**: AI 응답을 JSON으로 변환 및 노션 규격에 맞게 분할
7. **노션 저장**: 노션 API를 통해 블로그 DB에 페이지 생성

## 🛠️ 설정 및 환경변수
워크플로우 정상 작동을 위해 다음 설정이 필요합니다.

- **환경변수**:
    - `GEMINI_API_KEY`: Google AI Studio에서 발급받은 API 키 설정
- **API 헤더 (네이버)**:
    - `X-Naver-Client-Id`: 네이버 개발자 센터 발급 ID
    - `X-Naver-Client-Secret`: 네이버 개발자 센터 발급 Secret
- **노션 연동**:
    - 대상 DB ID: `39b2fdcea0a345a48db21f22a6d6b0e0`
    - 노션 DB의 'Connections' 설정에서 해당 워크플로우를 사용하는 인테그레이션이 활성화되어 있어야 합니다.

## ⚠️ 주의사항 및 트러블슈팅

### 1. API 인증 오류 (401)
- 네이버 API 키가 만료되었거나 오입력된 경우입니다. 네이버 개발자 센터에서 키를 재확인하세요.

### 2. Gemini API 호출 실패 (429/빈 응답)
- API 호출 한도(Rate Limit) 초과 시 발생합니다. 잠시 후 재실행하거나 Google AI Studio에서 사용량을 확인하세요.

### 3. 노션 저장 실패 (404/400)
- **404**: 노션 DB에 인테그레이션이 연결되지 않은 경우입니다. DB 설정에서 연결을 확인하세요.
- **400**: 본문 내용이 노션 블록 제한(2000자)을 초과한 경우입니다. '블로그 추출' 노드에서 청크 사이즈를 조정하거나 프롬프트의 자수 제한을 강화하세요.
