# 🤖 대치Math4U · DCT 업무자동화 n8n 워크플로우

n8n 기반 업무자동화 워크플로우 백업 및 문서화 저장소입니다.
직접 학원을 운영하면서 만든 워크플로우들을 제작/관리하는 repository로 Github를 이용하고 있습니다.
소규모 학원/교습소/공부방(개인지도 포함)을 운영하시는 분들이 참고하실 만한 워크플로우를 공개합니다.

각 폴더에 `workflow.json`(워크플로우 원본)과 `README.md`(노션 메뉴얼 연동 문서)가 자동 동기화됩니다.

---

## 📁 저장소 구조

```
/
├── README.md                          ← 이 파일
├── {워크플로우명}/
│   ├── workflow.json                  ← n8n 워크플로우 원본 (자동 동기화)
│   └── README.md                     ← 노션 메뉴얼 기반 문서 (자동 동기화)
└── ...
```

동기화는 `n8n Github Sync` 워크플로우가 수동 실행 시 전체 갱신합니다.

## 🔗 연동 서비스

- **Notion** — 학생 관리, 수업료 관리, 워크플로우 메뉴얼 DB
- **카카오 알림톡** — 상담/수업 관련 알림 발송
- **GitHub** — 워크플로우 백업 및 문서화 (이 저장소)
- **Gemini API** — 블로그 자동 작성 AI 모델

---

## 🔄 자동 동기화

이 저장소의 각 워크플로우 폴더는 `n8n Github Sync` 워크플로우가 실행될 때마다 자동으로 갱신됩니다.

- `workflow.json` — n8n API에서 직접 추출한 워크플로우 원본 (credentials 제거됨)
- `README.md` — Notion에 작성된 메뉴얼 페이지를 Markdown으로 변환

> 마지막 동기화 일시는 각 워크플로우 폴더의 README.md 상단에서 확인할 수 있습니다.
