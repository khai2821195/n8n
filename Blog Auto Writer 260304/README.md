# 📚 Math4U 학원 블로그 자동작성 워크플로우

이 워크플로우는 **대치Math4U 동탄캠**의 네이버 블로그 운영을 효율화하기 위해 설계된 자동화 도구입니다. 사용자가 웹 폼에 주제와 옵션을 입력하면, 참고 자료를 수집하고 AI를 통해 최적화된 블로그 초안을 자동으로 생성합니다.

---

## 🚀 워크플로우 목적
- 학원 블로그 포스팅 작성 시간 단축
- 타겟 독자(학부모, 학생)에 맞춘 전문적인 콘텐츠 생성
- 참고 URL 기반의 정보성 글 작성 지원

## ⚙️ 동작 방식 (Node Flow)

1.  **📝 블로그 작성 폼 (Form Trigger)**: 사용자가 주제, 대상 독자, 글 톤, 참고 URL, 추가 메모를 입력합니다.
2.  **입력값 파싱 (Set)**: 입력된 데이터를 워크플로우 내에서 사용하기 쉬운 변수 형태로 가공합니다.
3.  **URL 콘텐츠 수집 (Code)**: 입력된 참고 URL 리스트를 분리하고, Jina Reader API를 통해 읽기 좋은 텍스트 형태로 변환할 준비를 합니다.
4.  **Jina URL 읽기 (HTTP Request)**: `r.jina.ai` 서비스를 사용하여 외부 웹페이지의 텍스트 콘텐츠를 추출합니다.
5.  **URL 자료 통합 (Code)**: 수집된 여러 개의 URL 내용을 하나로 병합하여 프롬프트에 삽입할 준비를 합니다.
6.  **1차 프롬프트 조립 (Code)**: 수집된 자료와 사용자의 요청사항을 결합하여 AI 모델에 전달할 최종 프롬프트를 생성합니다.

---

## 🛠 필수 설정 및 환경
- **n8n 버전**: 1.0 이상 권장
- **외부 서비스**:
    - **Jina Reader**: 웹 콘텐츠 추출을 위해 사용됩니다. 별도의 API 키 없이도 URL 변환 기능을 활용합니다.
    - **AI 모델**: 워크플로우 후반부(미포함된 AI 노드 연결 시)에 OpenAI, Anthropic, 또는 Google Gemini 등의 LLM 연결이 필요합니다.

---

## ⚠️ 주의사항
- **URL 제한**: 참고 URL은 한 번에 여러 개 입력 가능하지만, 너무 많은 양의 데이터를 수집할 경우 토큰 제한이 발생할 수 있습니다.
- **데이터 처리**: `URL 자료 통합` 노드에서 각 URL당 최대 3,000자까지만 추출하도록 설정되어 있습니다.
- **결과물 확인**: 생성된 초안은 AI가 작성한 것이므로, 발행 전 반드시 학원 내부 가이드라인에 맞게 내용을 검토 및 수정하시기 바랍니다.

---

## 💡 사용 방법
1. n8n에 워크플로우를 임포트합니다.
2. 워크플로우를 활성화(Active)합니다.
3. 제공된 폼 URL에 접속하여 블로그 주제와 관련 정보를 입력합니다.
4. 워크플로우가 실행된 후, 연결된 AI 노드를 통해 생성된 초안을 확인합니다.
